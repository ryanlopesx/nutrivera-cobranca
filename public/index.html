Como funciona

1. Remetentes: Cadastre as instâncias do Evolution (números de WhatsApp). O sistema escolhe um aleatoriamente para cada envio.

2. Templates: Crie várias versões de mensagens de cobrança. Use {nome} para inserir o nome do cliente. O sistema sorteia um template diferente a cada envio.

3. Clientes: Adicione os clientes com telefone (DDD + número) e defina de quantas em quantas horas ele deve receber mensagens.

4. Automático: O sistema verifica a cada 60 segundos quem precisa receber mensagem e envia automaticamente. Para parar, clique em "Remover" na lista de clientes.
